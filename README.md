## Opis

Projekt polegał na stworzeniu dwóch banków (w różnych językach) oraz stworzenie API aby banki mogły się komunikować z Jednostką rozliczającą.

#### Technologie

- HTML5
- CSS
- PHP
- JavaScript
- Java (ThymeLeaf)
